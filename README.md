# Cursor IDE Setup Log

## Installed Tools
- **Cursor IDE** — AI-based editor, downloadable from https://cursor.com/
- **Claude Code Extension** — An additional AI assistant from Anthropic
- **Codex Extension** — An additional AI from OpenAI

## Completed Steps
1. Download and install Cursor IDE on your M1 MacBook
2. Install the Claude Code extension via the Extensions Marketplace
3. Log in to Claude Code using your Anthropic account
4. Install the Codex extension via the Extensions Marketplace
5. Create a GitHub account and a new public repository
6. Clone the repository locally and open it in Cursor
7. Create this README.md file
8. Commit and push to GitHub

## Problems & Solutions -

1. Confused about how to use cursors because they've never used them before? The solution is to watch a tutorial on YouTube and start practicing directly in the cursor application.
2. Unable to download.Git because there's not enough storage on the MacBook. This can be solved by deleting several unused applications.
3. Git isn't installed: macOS automatically prompts you to install Xcode Command Line Tools the first time you run the git command.
